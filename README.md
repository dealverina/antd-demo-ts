## Run Locally

Clone the project

```bash
  git clone https://github.com/dealverina/antd-demo-ts.git
```

Go to the project directory

```bash
  cd antd-demo-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
